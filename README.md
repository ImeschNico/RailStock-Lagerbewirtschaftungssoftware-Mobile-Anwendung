# RailStock-Lagerbewirtschaftungssoftware – Mobile Anwendung
Mobile Anwendung für die RailStock-Lagerbewirtschaftungssoftware.

## Proof of Concept
In einem kleinen Umfang wurde getestet, ob die mobile Anwendung Zugriff auf das [RailStock-Backend](https://github.com/ImeschNico/RailStock-Lagerbewirtschaftungssoftware-Backend.git) hat.

### Testergebnis
Die mobile Anwendung hat nach Anpassungen in den `application.properties` und dem `AndroidManifest.xml`-File erfolgreich Zugriff auf das Backend.  
Da der Proof of Concept gelungen ist, kann nun mit der Entwicklung der mobilen Anwendung mit vollständigen CRUD-Funktionen begonnen werden.
